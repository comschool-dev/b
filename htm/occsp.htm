Curso Oracle Cloud Commerce (OCC)

A demanda por profissionais qualificados e capacitados em OCC é enorme. O Curso de Oracle Cloud Commerce (OCC) foi desenvolvido para levar até você as principais melhores estratégias de gestão de uma das plataformas de ecommerce mais robustas do mercado.

Objetivos do curso Oracle Cloud Commerce (OCC):

• Preparar programadores para trabalhar com a OCC de forma ágil e efetiva
• Demonstrar as melhores ténicas de configuração da ferramenta
• Ensinar os atalhos para as principais funcionalidades e códigos do sistema.

Duração:

O curso Oracle Cloud Commerce tem carga horária total de 21 horas de muito conteúdo.

Público-Alvo do curso:

Programadores com conhecimento em Java
