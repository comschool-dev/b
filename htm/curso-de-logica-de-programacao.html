CURSO DE LÓGICA DE PROGRAMAÇÃO

O mercado de tecnologia é um dos que mais cresceram nos últimos anos, criando uma grande demanda de desenvolvedores capazes de implementar sistemas de qualidade  nas mais diversas áreas. E as perspectivas para o futuro são ainda melhores, uma vez que os sistemas estão cada vez mais presentes e complexos em todas as atividades econômica.

O conhecimento de lógica de programação é essencial na formação de um bom desenvolvedor independente da linguagem de programação que utilize, seja ela estruturada ou orientada a objeto. Por isso é o primeiro passo para se tornar um desenvolvedor.

IMPORTANTE: o curso é PRÁTICO. Você deve trazer seu notebook para acompanhar as aulas.

Objetivos do curso:

• Desenvolver o pensamento lógico;
• Entender como são feitos os sistemas de computador;
• Construir algoritmos com os elementos básicos de programação; 
• Preparar o aluno para aprender linguagens de programação

Duração do curso:

O Curso de Lógica de Programação tem carga horária total de 16 horas. 

Público-Alvo do curso:

Este curso é ideal para pessoas que desejam iniciar a carreira na área de desenvolvimento de sistemas.
